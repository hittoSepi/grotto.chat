# Channels

Channels are Grotto's group chats. Names always start with a hash, e.g. `#general`.

## Joining

Type `/join #channel`. If the channel doesn't exist, it's created automatically.

## Leaving

`/part` leaves the current channel.  
`/part #channel` leaves the named channel.

## Messaging

Type a message and press `Enter`. It goes to the active channel.

Switch channels with:
- `Alt+1..9`     jump to channel by number
- `Alt+←/→`      previous/next channel
- Click channel name with mouse

## User List

The right panel shows users on the channel. Drag the edge to adjust panel width.

Click a user to mention them in the input field.
