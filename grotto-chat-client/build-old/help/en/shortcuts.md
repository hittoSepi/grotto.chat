# Keyboard Shortcuts

## Basic Navigation

`Enter`           send message  
`Esc`             exit / close menu  
`Tab`             autocomplete command or nickname  
`PgUp / PgDn`     scroll message history up/down  
`Ctrl+V`          paste from clipboard  

## Channels

`Alt+1..9`        jump to channel 1–9  
`Alt+←`           previous channel  
`Alt+→`           next channel  

## Voice

`F1`              toggle push-to-talk (PTT) / voice activation (VOX)

## Views

`F2`              show/hide user list  
`F12`             open settings  

---

Tip: In PTT mode, hold `F1` down to talk.
