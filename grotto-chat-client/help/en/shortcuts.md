# Keyboard Shortcuts

## Basic Navigation

`Enter`           send message  
`Esc`             exit / close menu  
`Tab`             autocomplete command or nickname  
`PgUp / PgDn`     scroll message history up/down  
`Ctrl+V`          paste from clipboard  

## Channels

`Alt+1..9`        jump to channel 1–9  
`Alt+←`           previous channel  
`Alt+→`           next channel  

## Voice

configured PTT key (`F1` by default)  toggle PTT transmit on/off

## Views

`F2`              show/hide user list  
`F12`             open settings  

---

Tip: In PTT mode, press your configured PTT key again to stop transmitting.
