# Keyboard Shortcuts

## Basic Navigation

`Enter`           send message  
`Esc`             exit / close menu  
`Tab`             autocomplete command or nickname  
`PgUp / PgDn`     scroll message history up/down  
`Ctrl+V`          paste from clipboard  

## Channels

`Alt+1..9`        jump to channel 1–9  
`Alt+←`           previous channel  
`Alt+→`           next channel  

## Voice

configured talk key                    toggle talk in Toggle to Talk mode, or hold-to-talk in Push to Talk mode

## Views

`F2`              show/hide user list  
`F12`             open settings  

---

Tip: `Toggle to Talk` stays live until you press the talk key again. `Push to Talk` stays live only while the key is held.
