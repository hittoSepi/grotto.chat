# Grotto Help

Welcome to the cave. The server doesn't read your messages here.

Type `/help <topic>` if you want to know more:

- `commands`   - All commands
- `shortcuts`  - Keyboard shortcuts
- `channels`   - Using channels
- `voice`      - Voice calls
- `crypto`     - Encryption and trust
- `files`      - File transfers

---

## Quick Guide

`Enter`        send message  
`Tab`          autocomplete name/command  
`PgUp/PgDn`    scroll history  
`Alt+1..9`     jump to channel  
`Alt+←/→`      previous/next channel  
`F12`          settings  

First steps after login:

1. Join a channel: `/join #general` (or `/j #general`)  
2. List users: `/names` (or `/ns`)  
3. Send DM: `/msg <name> <text>` (or `/w <name> <text>`)  
4. Need command list: `/help commands`  

Most common commands:

`/join #channel`    join a channel  
`/part`             leave channel  
`/msg <name> ...`   private message  
`/call <name>`      start voice call  
`/quit`             close Grotto  

## Mouse

Yeah, the mouse works in the terminal:

- Click a channel to switch
- Click a user to mention them
- Drag the right edge to resize the panel
- Double-click a message to select it
- Triple-click copies the whole message to clipboard
